## User Summary

The rollout trims every `lwndev-sdlc` sub-skill's `SKILL.md` to the same lite-style dispatcher shape piloted in `orchestrating-workflows` (CHORE-035). Twelve target skills across the feature, chore, and bug workflow chains receive prose compression and have heavy narrative (long numbered recipes, multi-step procedures, verbose decision flows) relocated to per-skill `references/` directories with single-sentence inline pointers left in place. The goal is to reduce per-invocation input-token surface consistently while preserving every load-bearing carve-out established by CHORE-034 and the `## Output Style` sections installed by FEAT-023. Baseline and post-change `wc` measurements are captured per skill and aggregated into a grand-total delta table in the requirements Notes section. The rollout is documentation/style only — no runtime behavior changes and `npm run validate` / `npm test` must continue to pass unchanged.
